# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- MCP Remote Proxy mode (`--wrap`) for wrapping remote MCP servers with P2P transport

## [0.7.3] - 2026-03-21

### Fixed

- Switch to OIDC Trusted Publishing for PyPI and npm releases

## [0.7.2] - 2026-03-21

### Fixed

- Resolve ruff lint errors and add smoke tests

## [0.7.1] - 2026-03-20

### Added

- Initial release — MCP server for P2P agent networking
- CI/CD workflows for automated testing and publishing
- Platform configurations for Cursor, Windsurf, Cline, and Roo Code
- Support for 13+ AI tool platforms via `uvx agentanycast-mcp`
